The **Widget3D** is the 3D representation of a <Classes.Widget /> in the world! As it is an Actor, you have all the flexibility of it's methods!


## 🎒 Examples {/* #examples */}

#### Spawning a Widget3D from a Widget {/* #spawning-a-widget3d-from-a-widget */}

```lua title="Client/Index.lua" showLineNumbers
-- Spawns a Widget
local my_text = Widget(NativeWidget.Text)
my_text:CallBlueprintEvent("SetText", "Hello World!")

-- Spawns the Widget3D directly from a Widget
local my_widget_3d = my_text:SpawnWidget3D()
```
