### `SetHighlightColor`

> Changes the Highlight Color for highlighted actors at a specific Index. Multiply it by 5 \(or more\) for having a glowing effect.
> 
> `HighlightMode.Always` will always be visible, even behind walls

### `Trace`

> Trace a ray against the world and returns a table with the first blocking hit information. Currently only supports trace for <Classes.Character />, <Classes.Vehicle />, <Classes.StaticMesh />, <Classes.Weapon />, <Classes.Item />, <Classes.Grenade /> and <Classes.Prop />, more can be requested.
>
> Returned Table: `{"Success", "Location", "Normal", "Entity", "BoneName", "ActorName", "ComponentName", "SurfaceType"}`

```lua
Client.Trace(start_location, end_location, collision_channel, trace_complex, return_entity, return_physical_material, draw_debug)
```

| Type | Parameter | Default Value | Description |
| :--- | :--- | :--- | :--- |
| <Structs.Vector /> | **`start_location`** |  |  |
| <Structs.Vector /> | **`end_location`** |  |  |
| <Enums>CollisionChannel</Enums> | **`collision_channel`** | `CollisionChannel.WorldStatic` | Supports several channels separating by `|` |
| <BasicType.Boolean /> | **`trace_complex`** | `false` |  |
| <BasicType.Boolean /> | **`return_entity`** | `false` |  |
| <BasicType.Boolean /> | **`return_physical_material`** | `false` |  |
| <BasicType.Boolean /> | **`draw_debug`** | `false` |  |
