Integração do Discord

Como usar nanos métodos do Discord para criar uma boa integração para o seu servidor.

Integrar seu servidor ao Discord é muito fácil! Nós fornecemos uma integração que está pronta para ser usada, mas você também pode adicionar seu próprio ID de aplicativo do Discord e usar seu próprio aplicativo!

Atualizando a atividade do jogador do Discord (Rich Presence)​

Fizemos o processo muito simples, temos um método para alterar a atividade do jogador:

Client/Index.lua
local state = "Working as Truck Driver"
local details = "Awesome Roleplay"
local large_text = "Honk!"
local large_image = "nanos-world-full-world"

Client.SetDiscordActivity(state, details, large_image, large_text)

info

Você pode usar seu próprio ID de Aplicativo para enviar suas próprias imagens!

Usando seu próprio ID de Aplicativo​

Usar o seu próprio ID de Aplicativo traz a vantagem de ser capaz de configurá-lo como você quer, e até mesmo enviar suas próprias imagens para serem exibidas no Rich Presence.

Inicializar seu próprio ID de aplicativo é extremamente fácil, basta executar o seguinte código do cliente:

Client/Index.lua
Client.InitializeDiscord(client_id)
tip

E você está pronto! Agora você pode usar o método Client.SetDiscordActivity()!
