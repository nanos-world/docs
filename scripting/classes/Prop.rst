****
Prop
****

A Prop represents a Static Mesh which can be spawned in the world, can be grabbed around and have physics.

.. attention:: This page is under construction.


.. tip:: You can see all the default static meshes in `<NanosWorldFolder>\NanosWorld\Content\NanosWorld\`


Usage
-----

.. tabs::
 .. code-tab:: lua Lua

    local MyProp = Prop(
      Vector(-900, 185, 215), 
      Rotator(0, 90, 90), 
      "NanosWorld/Art/Rural/Extra/SM_Crate_07"
    )
