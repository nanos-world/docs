ğŸ”£ Blueprint

A Blueprint Class allows spawning any Unreal Blueprint Actor in nanos world.

info

ğŸŸ¦ğŸŸ§ Authority: This class can be spawned on both Client and Server (if you spawn it on client, it won't be synchronized with other players).

â¬ Inheritance: This class inherits from Actor and Paintable.

Usageâ€‹

Index.lua
local my_blueprint = Blueprint(
    Vector(200, 200, 200), -- location
    Rotator(0, 0, 0), -- rotation
    "nanos-world::BP_AwesomeDoor" -- blueprint
)
tip

If your Actor Blueprint was spawned on the Server, it will be automatically synchronized with other players using the nanos world Network Authority system! It follows the same rules as all other entities!

note

Currently it is only possible to communicate in one-way with the Blueprint (Scripting -> Blueprint). We didn't find a way to have the inverse communication hopefully yet.
