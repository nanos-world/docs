You can use some Unreal Engine plugins when creating an asset pack for nanos world. You can learn more about UE Plugins [here](https://dev.epicgames.com/documentation/en-us/unreal-engine/plugins-in-unreal-engine).


## Whitelisted Unreal Engine Plugins

### External Plugins

You need to install it manually and/or have purchased it.

- [Kawaii Physics](https://github.com/pafuhana1213/KawaiiPhysics)
- [Ultra Dynamic Sky](https://fab.com/s/77a632ad8b62)


### Native Plugins

You can just enable them in Plugins panel.

- [Pose Search](https://dev.epicgames.com/documentation/en-us/unreal-engine/BlueprintAPI/Animation/PoseSearch)
- [Animation Warping](https://fab.com/s/e9829de17f28)
- [Motion Warping](https://dev.epicgames.com/documentation/en-us/unreal-engine/motion-warping-in-unreal-engine)
- [Chooser](https://dev.epicgames.com/documentation/en-us/unreal-engine/API/Plugins/Chooser)
- Motion Trajectory
- [Chaos Vehicles](https://dev.epicgames.com/community/learning/tutorials/ZeKz/unreal-engine-setting-up-the-chaos-vehicles-plugin)
- [PCG (Procedural Content Generation)](https://dev.epicgames.com/community/learning/tutorials/j4xJ/unreal-engine-introduction-to-procedural-generation-plugin-in-ue5-2)
- [HDRI Backdrop](https://dev.epicgames.com/documentation/en-us/unreal-engine/hdri-backdrop-visualization-tool-in-unreal-engine)
- [CommonUI](https://dev.epicgames.com/documentation/en-us/unreal-engine/overview-of-advanced-multiplatform-user-interfaces-with-common-ui-for-unreal-engine)
- [ModelViewViewModel (MVVM)](https://dev.epicgames.com/documentation/en-us/unreal-engine/umg-viewmodel-for-unreal-engine)
- [Sun Position Calculator](https://dev.epicgames.com/documentation/en-us/unreal-engine/sun-and-sky-actor-in-unreal-engine)
- [Water Plugin](https://dev.epicgames.com/documentation/en-us/unreal-engine/water-system-in-unreal-engine)
- [Geometry Scripting](https://dev.epicgames.com/documentation/en-us/unreal-engine/geometry-scripting-users-guide-in-unreal-engine)
- Nanite Displaced Meshes
- Rig Logic (MetaHuman)
- [LiveLink Control Rig](https://dev.epicgames.com/documentation/en-us/metahuman/animating-metahumans-with-livelink-in-unreal-engine)
- Animation Locomotion Library
- Hair Strands
