🤝 Contribuer à la documentation

Bonjour Dyno! Bonjour Markdown!

Cette documentation est faite sur la plateforme Docusaurus et est écrite en Markdown! N'hésitez pas à suggérer des modifications et à modifier les pages sur GitHub, cette aide est vraiment important pour nous et nous apprécions réellement ceux qui aident à l'écrire !

Suggérer des Modifications​

Vous pouvez suggérer des changements directement sur GitHub en appuyant sur le bouton Edit this Page en bas à gauche de chaque page, faites juste une suggestion et ouvrez une Pull Request dessus ! Une fois acceptée, votre suggestion est immédiatement appliquée à la version bleeding-edge des docs !

Référence de l'API​

Nos Docs sont intelligents ! La section Référence de script est automatiquement générée à partir de notre API Repository JSON sur GitHub. Nous avons défini toutes les classes, méthodes et événements en JSON qui peuvent être analysés dans n'importe quel format !

Donc si vous voulez suggérer un changement à une description ou une faute de frappe que vous avez trouvée, reportez-vous à nos fichiers de l'API Repository: Classes, StaticClasses, Structs, UtilityClasses, StandardLibraries ou Enums !

Traduction des pages​

Par défaut, toutes les pages sur GitHub sont en anglais et toutes les traductions proviennent de Crowdin. Vous pouvez accéder à la documentation officiel de nanos sur Crowdin et nous aider à réaliser la traduction pour toutes les langues disponibles ! Vous pouvez demander d'autres langues ! Merci de nous faire savoir si vous voulez nous aider dans une nouvelle langue !
